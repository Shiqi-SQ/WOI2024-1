# 排名规则
本次比赛采用 IOI 赛制：**选手排名根据选手的题目得分和作答用时共同计算**。每题分为多个测试点，通过部分测试数据将得到此题的部分分数，通过所有测试数据将得到此题的全部分数，**题目得分以最后一次提交为准**。在多名选手答题得分相同的情况下，作答总用时短的选手排名将会靠前。
