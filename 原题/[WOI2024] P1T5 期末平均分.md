# U413157 [WOI2024] P1T5 期末平均分

## 题目描述

小卢的期末考试成绩出来了，语文、数学、英语 三门学科的成绩均已公布，现在小卢想请你帮他编写一个程序，计算 $3$ 门学科的总平均分（小卢只需知道平均分的整数部分即可）。

## 输入格式

输入三个整数 $a, b, c$ 分别代表三门学科的成绩（$0 < a, b, c \le 100$）。

## 输出格式

输出一个整数，代表计算得到的整数平均分。

例如：  
- $70.23$ 只取 $70$  
- $95.99$ 只取 $95$

## 输入输出样例 #1

### 输入 #1

```
80 90 95
```

### 输出 #1

```
88
```
